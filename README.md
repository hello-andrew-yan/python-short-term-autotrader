<h3 align="center">Motivation</h3

While studying short-term trading strategies, a recurring observation was that proficient traders rely heavily on pattern recognition in conjunction with disciplined risk management. This raised a question worth investigating:

> If the edge in short-term trading is fundamentally pattern-based, can a machine learning model learn those patterns directly from price data and replicate that edge without domain-specific discretion?

This repository is an attempt to answer that empirically.
